# 작업 로그 - Button 컴포넌트 구현

## 작업 일자
2025-11-09

## 작업 개요
Figma 디자인을 기반으로 Button 컴포넌트를 구현하고 Storybook에 모든 배리에이션을 추가했습니다.

## 주요 작업 내용

### 1. Button 컴포넌트 구현 (`components/ui/button/Button.tsx`)

#### 컴포넌트 구조
- **기반 라이브러리**: Ant Design의 `Button` 컴포넌트를 래핑
- **아이콘 라이브러리**: `@heroicons/react` 사용

#### 타입 정의
```typescript
export type ButtonType = 'default' | 'plain' | 'outline'; // Figma: Default, Plain, Outline
export type ButtonSize = 'xs' | 'sm' | 'base' | 'l' | 'xl'; // Figma: xs, sm, base, l, xl
export type ButtonState = 'default' | 'hover' | 'focus' | 'disabled'; // Figma: Default, Hover, Focus, Disabled
```

#### 주요 기능
1. **타입별 스타일링**
   - `default`: Primary 버튼 (어두운 배경, 흰색 텍스트)
   - `plain`: Secondary/Ghost 버튼 (밝은 배경, 어두운 텍스트, **border 없음**)
   - `outline`: Outline 버튼 (투명 배경, border 있음)

2. **사이즈별 border-radius**
   - `xs`, `sm`: 6px (`rounded-md`)
   - `base`, `l`, `xl`: 8px (`rounded-lg`)

3. **Hover 상태**
   - 모든 타입에 2.5% 검은색 오버레이 적용
   - `box-shadow: inset 0 0 0 9999px rgba(0,0,0,0.025)` 사용

4. **Focus 상태**
   - 파란색 ring 적용 (`ring-2 ring-border-extensions-border-info`)

5. **Disabled 상태**
   - `default`: 회색 배경, 회색 텍스트
   - `plain`: 투명 배경, 회색 텍스트
   - `outline`: 투명 배경, 회색 텍스트, 회색 border

6. **아이콘 지원**
   - `iconOnly` prop으로 아이콘만 표시 가능
   - 텍스트와 아이콘 함께 표시 가능
   - `heroicons` 아이콘 사용

### 2. Storybook 구현 (`components/ui/button/Button.stories.tsx`)

#### Light Mode 배리에이션 (10 rows × 8 columns)
1. **Row 1**: Default Type, Default State
   - Icon only (xs, sm, base)
   - Text only (xs, sm, base, l, xl)

2. **Row 2**: Outline Type, Default State
   - Icon only (xs, sm, base)
   - Text only (xs, sm, base, l, xl)

3. **Row 3**: Default Type, Hover/Focus State
   - Focus ring 적용

4. **Row 4**: Default Type, Disabled State

5. **Row 5**: Plain Type, Disabled State

6. **Row 6**: Plain Type, Default State

7. **Row 7**: Outline Type, Primary Color, Default State
   - 파란색 border와 텍스트 (`border-border-extensions-border-info`)

8. **Row 8**: Plain Type, Default State, Larger
   - 사이즈 l, xl 사용

9. **Row 9**: Outline Type, Primary Color, Default State, Larger
   - 사이즈 l, xl 사용

10. **Row 10**: Plain Type, Disabled State (반복)

#### Dark Mode 배리에이션 (9 rows × 10 columns)
1. **Row 1**: Subtle Default State
   - Icon only (xs, sm, base, l, xl)
   - Text only (xs, sm, base, l, xl)

2. **Row 2**: Default/Hover State

3. **Row 3**: Active/Focused State
   - Focus ring 적용

4. **Row 4**: Light Disabled/Ghost State
   - 투명도 50% 적용

5. **Row 5**: Medium Disabled/Ghost State
   - 투명도 30% 적용

6. **Row 6**: Emphasized Default/Hover State

7. **Row 7**: Active/Focused - Emphasized State
   - Focus ring 적용

8. **Row 8**: Heavy Disabled/Ghost State
   - 투명도 20% 적용

9. **Row 9**: Text-Only / Fully Disabled State
   - 투명도 10% 적용

### 3. 수정 사항

#### 수정 1: Border-radius 적용
- **문제**: border-radius가 적용되지 않음
- **해결**: 사이즈별로 `rounded-md` (6px) 또는 `rounded-lg` (8px) 적용
- **방법**: `getBorderRadius` 함수로 동적 클래스 생성 + inline style로 값 명시

#### 수정 2: Hover/Focus 상태
- **문제**: Hover/Focus 상태가 Figma 디자인과 다름
- **해결**: 
  - Hover: `box-shadow: inset 0 0 0 9999px rgba(0,0,0,0.025)` 사용
  - Focus: `ring-2 ring-border-extensions-border-info` 적용

#### 수정 3: Light Mode Primary Button 배경색
- **문제**: Primary 버튼 배경색이 잘못됨
- **해결**: `bg-background-background-inverse-primary` 사용 (어두운 배경)

#### 수정 4: Plain 타입에서 Border 제거
- **문제**: Plain 타입 버튼에 border가 들어감
- **해결**: 
  - Button 컴포넌트에서 `border-0` 클래스 적용
  - Storybook의 Dark 모드에서 추가한 `border` 클래스 모두 제거

#### 수정 5: 임의로 추가한 부분 제거
- **문제**: Row 8-9에 `text-lg font-bold`, `text-xl font-bold`, `text-2xl font-bold` 임의 추가
- **해결**: 이미지 설명에 "약간 더 크거나 굵게"만 언급되어 있어 제거

## 사용된 디자인 토큰

### 색상 토큰
- `--background-background-inverse-primary`: Primary 버튼 배경 (어두운 색)
- `--background-background-secondary`: Plain 버튼 배경 (밝은 색)
- `--content-content-inverse-primary`: Primary 버튼 텍스트 (흰색)
- `--content-content-primary`: Plain/Outline 버튼 텍스트 (어두운 색)
- `--border-extensions-border-info`: Focus ring 및 Primary outline 색상 (파란색)
- `--border-border-primary`: Outline 버튼 border 색상

### Border Radius 토큰
- 사이즈별로 직접 값 사용 (6px 또는 8px)

## 기술 스택

- **Storybook**: 컴포넌트 문서화 및 시각화
- **@heroicons/react**: 아이콘 라이브러리
